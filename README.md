# Lugx - Gaming Website

Lugx is a modern gaming website template built with **HTML**, **CSS**, and **Tailwind CSS**.  
It showcases trending games, most played games, top categories, and a functional cart system.

---

## Features

- Responsive design for mobile, tablet, and desktop.
- Trending, Most Played, and Top Categories sections.
- Interactive game cards with "Add to Cart" functionality.
- Sticky navigation bar.
- Animated hover effects.
- Modern UI with Tailwind CSS utility classes.

---

## Technologies Used

- [HTML5](https://developer.mozilla.org/en-US/docs/Web/HTML)
- [CSS3](https://developer.mozilla.org/en-US/docs/Web/CSS)
- [Tailwind CSS](https://tailwindcss.com/)
- [Font Awesome](https://fontawesome.com/) icons
- Vanilla JavaScript for interactivity
